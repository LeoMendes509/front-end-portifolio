Sobre mim

Olá, meu nome é Leonardo Mendes Rodrigues, sou estudante de Análise e Desenvolvimento de Sistemas e um entusiasta da tecnologia. Tenho interesse em desenvolvimento Full Stack, buscando sempre aprimorar minhas habilidades em Front-end e Back-end para criar soluções tecnológicas eficientes e inovadoras.

Atualmente, possuo conhecimento em desenvolvimento tanto no lado do cliente (Front-end) quanto no lado do servidor (Back-end), utilizando tecnologias modernas e frameworks amplamente adotados no mercado. Front-end HTML5, CSS3 e JavaScript Frameworks e bibliotecas: React.js Gerenciamento de estado e manipulação do DOM Responsividade e acessibilidade Consumo de APIs REST Back-end Linguagem Java Frameworks: Spring Boot, Hibernate Desenvolvimento de APIs REST Banco de dados relacionais (MySQL, PostgreSQL) Autenticação e segurança Linguagens e Frameworks Java: Uma das linguagens mais utilizadas no mercado, com foco em aplicações robustas e escaláveis. Spring Boot: Framework para criação de aplicações Java eficientes. Hibernate: Framework ORM para manipulação de banco de dados. JavaScript: Fundamental para o desenvolvimento web moderno. React.js: Biblioteca para criação de interfaces dinâmicas e reativas. Estou sempre em busca de aprendizado e desafios na área de tecnologia. Meu objetivo é me tornar um desenvolvedor Full Stack completo, combinando habilidades de Front-end e Back-end para criar aplicações de alto desempenho. Estou aberto a oportunidades de aprendizado e desenvolvimento profissional! Obrigado!
